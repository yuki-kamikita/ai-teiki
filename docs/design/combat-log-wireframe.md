## 戦闘ログ詳細ページ（ワイヤー）

```
┌──────────────────────────────────────────────┐
│ 戦闘ログ詳細                                  │
├──────────────────────────────────────────────┤
│ [1] 基本情報                                  │
│ 日時 / 対戦相手 / 勝敗                         │
├──────────────────────────────────────────────┤
│ [2] 結果サマリー                               │
│ 残りHP（自分/相手）                            │
│ 獲得報酬（装備/スキル/称号/実績）               │
│ 状態異常の要約（付与/解除）                     │
├──────────────────────────────────────────────┤
│ [3] 使用スキルの要約                           │
│ 最後の1〜3件のスキル名                          │
├──────────────────────────────────────────────┤
│ [4] ターンログ（詳細）                          │
│ ターン1: 行動順 / 使用スキル / ダメージ / 回復  │
│        状態異常 / クリティカル / 回避 / HP/MP   │
│ ターン2: ...                                   │
│ ...                                            │
└──────────────────────────────────────────────┘
```

### 表示ルール
- 1画面で見やすい範囲を表示し、長い場合はスクロール
- ターンログは時系列で並べる
