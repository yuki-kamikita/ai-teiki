## 戦闘ログ仕様（開発者・AI向け）

### 一括表示（サマリー）に含める情報
- 勝敗
- 対戦相手の名前
- 自分/相手の残りHP
- 使用した主なスキル（最後の1〜3件）
- 付与/解除された状態異常
- 獲得報酬（装備/スキル/称号/実績）

### 画面構成
- ログ一覧は作らない
- プレイヤー画面から「戦闘ログ詳細」ページへ直接遷移する
- 詳細は別ページで表示する

### ログ詳細ページの並び（上から）
1. 基本情報（日時・対戦相手・勝敗）
2. 結果サマリー（残りHP・獲得報酬・状態異常の要約）
3. 使用スキルの要約（最後の1〜3件）
4. ターンログ（詳細）

### 詳細ログ（ターン単位）
- ターン番号
- 行動順
- 使用スキル
- 与ダメージ/回復量
- 付与/解除された状態異常
- クリティカル/回避の結果
- HP/MPの変化

### 履歴
- 保存期間: 無期限

### 公開範囲
- 全体公開
